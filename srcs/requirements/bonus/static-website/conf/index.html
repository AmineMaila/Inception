Inception Project

42 Network DevOps Learning Journey

Project Overview

Inception is a challenging DevOps project in the 42 network curriculum that focuses on system administration and containerization. The primary goal is to broaden students' understanding of system configuration and Docker technologies.

Project Objectives

Key objectives of the Inception project include:

  • Learn how to use Docker to containerize various services
  • Create a complete web infrastructure using Docker Compose
  • Set up multiple services like WordPress, MariaDB, and NGINX
  • Understand network configuration and service communication
  • Practice system administration and security best practices

Technical Stack

Docker Docker Compose NGINX MariaDB WordPress Linux

Key Learning Outcomes

By completing the Inception project, students gain practical experience in:

  • Containerization techniques
  • Service orchestration
  • Network configuration
  • Security implementation
  • System administration
